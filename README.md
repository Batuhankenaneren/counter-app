# Counter App

A simple counter application built with **vanilla JavaScript**, inspired by the classic Passenger Counter tutorial and extended with additional logic and UI/UX improvements.

## Features
- Increment and decrement buttons (+ / -)
- Prevents negative values
- Save counter values to history
- Reset current counter
- Clear history with confirmation
- Undo last saved entry
- Responsive button layout
- Background image with blur effect

## Improvements over the original tutorial
- Added decrement functionality
- Added reset and clear separation
- Confirmation dialog for destructive actions
- Improved layout and styling
- Better state handling

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

## Notes
This project is based on a common beginner tutorial and was extended to explore better UX decisions and application logic.
